All new Wordpress Woocommerce plugin by which a customer can group all reviews by its ratings.

== Description ==

This is a woocommerce plugin using which you can sort all reviews by ratings. This plugin will allow you to add a dropdown box at review tab from which a customer can sort reviews by selected rating.

= Plugin Features =

*   This plugin have a settings page from where user can easily turn on/off it's functionality.
*   This plugin also allows to override the labels of the dropdown on review page simply by using shortcodes.

= Minimum Requirements =

*   Wordpress 3.6 or later
*   Woocommerce 2.1 or greater

== Installation ==

1.  Upload your plugin folder to the '/wp-content/plugins' directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Turn on the plugin's activity through the "Product Review Sorting Settings" menu on your Wordpress admin panel.
4.  All done. Just check on product review tab and group reviews according to its rating.
